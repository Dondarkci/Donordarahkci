# **App Name**: KCI Blood Drive Registration

## Core Features:

- GitHub Authentication: Secure user authentication via GitHub accounts.
- Registration Form: Collect user details: Full Name, NIK (KTP), WhatsApp number.
- Location & Date Selection: Allow users to select a location and date from available options with limited slots.
- Realtime Quota Management: Dynamically update and display available slots, disabling options once full.
- Success Notification: Display a 'Registration Successful' notification upon successful registration.
- WhatsApp Confirmation: Send a confirmation message via WhatsApp with registration details, including the chosen location and date.
- Data Export: Enable admin to download the user data as excel (.xlsx) file

## Style Guidelines:

- Primary color: Deep crimson (#8B0000), evoking the life-saving impact of blood donation and the corporate KAI color identity.
- Background color: Soft, desaturated rose (#FAEBD7), for a comforting and supportive atmosphere.
- Accent color: Warm beige (#DEB887), to highlight key actions and information.
- Headline font: 'Belleza' sans-serif, to bring a balance of fashion and artistic impression
- Body font: 'Alegreya' serif, for an elegant and readable style, and should longer text blocks be required.
- Use minimalist icons for form fields and location options, such as a heart with a blood drop.
- Subtle transitions on button interactions and form confirmations to enhance user experience.
